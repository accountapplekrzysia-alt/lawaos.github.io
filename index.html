Nowoczesny system operacyjny startujący do GNOME, z możliwością wyboru środowiska podczas instalacji.

Pobierz Jak działa instalator

Czym jest LawaOS?

LawaOS to autorski system operacyjny tworzony przez Krzysztofa. System startuje do środowiska GNOME, aby zapewnić wygodny i nowoczesny interfejs od pierwszego uruchomienia.

Podczas instalacji użytkownik może wybrać inne środowisko graficzne, takie jak KDE Plasma, XFCE, Cinnamon lub pozostać przy GNOME.

Najważniejsze funkcje

Start do GNOME

System uruchamia się bezpośrednio do GNOME, zapewniając gotowe środowisko od razu po starcie ISO.

Wybór środowiska

Instalator pozwala wybrać dowolne środowisko graficzne przed instalacją systemu.

Modularna budowa

System jest projektowany modułowo, z własnym formatem paczek .lawa.

Nowoczesny wygląd

Domyślny motyw LawaOS jest ciemny, minimalistyczny i inspirowany kolorem lawy.

Instalator LawaOS

Instalator działa w środowisku GNOME i prowadzi użytkownika przez proces instalacji krok po kroku.

  • Wybór środowiska graficznego
  • Wybór dysku i partycjonowanie
  • Instalacja systemu i paczek
  • Konfiguracja użytkownika
